Windows常用dos命令

2018-11-14 Windows技巧 阅读量

echo %变量名%查看某个系统变量的值

cd命令进入指定盘符（绝对路径）

  • -d 指定路径

dir 查看目录及文件列表

cls 清理屏幕

tree查看目录树

start 打开一个新的dos窗口或者打开本地当前所在目录
